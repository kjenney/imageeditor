# Image Editor

A modern, web-based image editor built with React and TypeScript.

## Overview

Image Editor is a powerful, browser-based image editing application designed with modern web technologies. It provides an intuitive interface for image manipulation with professional-grade features.

## Key Features

- **Modern Tech Stack** - Built with React 19, TypeScript, and Vite for optimal performance
- **Layer Support** - Work with multiple image layers for complex compositions
- **Extensible Architecture** - Modular design for easy feature additions
- **Fast Performance** - Optimized build process with code splitting
